# AMMEX Case Study Page

## Overview
Custom JavaScript for a Webflow page, published to npm as `@aw-webflow/ammex_case_study_page` and consumed via jsDelivr CDN.

## Code Style
- All code must use `var` and ES5 syntax for maximum browser compatibility
- No arrow functions, template literals, let/const, or other ES6+ features

## Deployment
1. Push changes to GitHub
2. Bump version in `package.json`
3. Run `npm publish --access public`
4. jsDelivr automatically serves the latest published version
